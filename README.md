# CSS-Based-Spotify-Clone
A music player UI inspired by Spotify, built using HTML and CSS, with features including responsive design, sidebar navigation, music player controls, and dark theme styles.
Spotify Clone :- 
A pixel-perfect Spotify UI clone built using pure HTML and CSS. This project replicates the core visual design of Spotify's web interface, including the sidebar, main content area, and music player bar — all without any JavaScript frameworks.

Features:

-> Dark-themed UI inspired by Spotify's design system
-> Responsive layout using Flexbox/Grid
-> Music player bar with playback controls
-> Sidebar with navigation links and playlists
-> Album/playlist card grid layout

Built With
  HTML5
  CSS (Flexbox / Grid)
